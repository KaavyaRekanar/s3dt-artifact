# Artifact for CCS Submission (Anonymized)

This repository contains artifacts for the paper:
**"Semantic Integrity as a Security Primitive: Detecting Drift in Digital Threads with LLM-Assisted Semantic Analysis"**

## Contents

- **S3DT_framework.ipynb**  
  Demonstrates the core components of the S3DT framework, including:
  - semantic claim representation  
  - digital thread construction  
  - drift detection and scoring  

- **Anthropic_case_study_setup.ipynb**  
  Reconstructs the case study used in the paper and demonstrates:
  - artifact modeling  
  - semantic claim extraction  
  - drift analysis across lifecycle stages  

## Usage

1. Open the notebooks in:
   - Jupyter Notebook, or  
   - Google Colab   

2. Run all cells sequentially to reproduce:
   - semantic representations  
   - drift detection outputs  
   - example drift scores  

## Notes

- The notebooks are self-contained demonstrations of the methodology described in the paper.
- They are intended to illustrate the semantic modeling and drift detection pipeline rather than a production system.
- All artifacts have been anonymized for double-blind review.
- If GitHub preview fails, please open notebooks in Google Colab or download and run locally.

## Requirements

The notebooks rely on standard Python libraries (e.g., NumPy).  
No special setup is required beyond a typical Python/Colab environment.
